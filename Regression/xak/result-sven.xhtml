Calculator Documentation

The Calculator provides basic functionality for processing arithmetic operations. The Calculator has the following features:
  • A core component for handling arithmetic expressions
  • A command line parser for using the Calculator from the console
  • A GUI component as graphical frontend of the Calculator
  • Several arithmetic operations are supported:
    • Addition

Overview of the Supported Features

Calculator Core Component

The core component is the basis of the Calculator. It provides basic functionality for handling arithmetic expressions.

Command Line Parser

The commend line parser provides a frontend for accessing the Calculator functionality from the console. The parser is generated automatically from a Bali grammar.

GUI

The GUI component provides graphical access to the Calculator.

Supported Arithmetic Operations

Addition

The Calculator supports addition of arithmetic terms.

Subtraction

The Calculator supports subtraction of arithmetic terms.

Multiplication

The Calculator supports multiplication of arithmetic terms.

Division

The Calculator supports division of arithmetic terms.
